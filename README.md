# 仿小红书 uni-app APK 壳项目

这是给 `springboot_vue_xhs-master` 配套整理的 HBuilderX/uni-app 离线演示版项目。它可以被 HBuilderX 识别并打包 APK，不需要启动 Spring Boot、MySQL、Redis。

## 使用步骤

1. 用 HBuilderX 打开本目录。
2. 在 HBuilderX 里选择 `运行 -> 运行到手机或模拟器 -> 运行到 Android App 基座` 先测试。
3. 确认没问题后，选择 `发行 -> 原生 App-云打包` 打 APK。

## 离线功能

- 首页展示本地模拟笔记。
- 登录页随便输入手机号，验证码固定为 `1234`。
- 发布页可以选择图片、填写标题和正文，数据保存到本地缓存。
- 回到首页会显示刚发布的本地笔记。
- 底部为首页、市集、发布、消息、我的五项导航。
- 未登录点击“我”会自动跳出登录页，登录后显示个人主页。

## 说明

原仓库没有包含作者的完整 uni-app 移动端源码，也没有 graceUI 授权文件。这个目录是可打包 APK 的离线演示版，适合交作业、演示界面和打安装包。
